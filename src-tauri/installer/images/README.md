# 安装程序图片资源说明

本目录存放 Windows NSIS 安装程序所需的图片资源。

## 所需图片

### 1. installer-header.bmp
- **用途**: 安装向导顶部横幅图片
- **尺寸**: 推荐 500 x 60 像素
- **格式**: BMP 格式
- **说明**: 显示在安装程序的顶部，通常包含应用程序名称或 Logo

### 2. installer-sidebar.bmp
- **用途**: 安装向导左侧侧边栏图片
- **尺寸**: 推荐 164 x 314 像素
- **格式**: BMP 格式
- **说明**: 显示在安装程序的左侧，可以放置应用图标或装饰图案

## 如何准备图片

### 方法 1: 使用在线工具
1. 访问 https://www.canva.com/ 或其他设计工具
2. 创建对应尺寸的画布
3. 添加应用名称、Logo 和装饰元素
4. 导出为 PNG 格式
5. 使用图片转换工具转换为 BMP 格式（如 https://www.online-convert.com/）

### 方法 2: 使用示例图片
如果暂时不需要自定义图片，可以使用纯色背景：
1. 创建对应尺寸的纯色图片
2. 添加简单的文字或 Logo
3. 转换为 BMP 格式

### 方法 3: 使用 Photoshop/GIMP
1. 新建画布（对应上述尺寸）
2. 设计安装程序界面
3. 另存为 BMP 格式

## 图片要求

- **颜色深度**: 24位或32位
- **文件格式**: BMP (Windows Bitmap)
- **文件大小**: 建议不超过 500KB
- **背景**: 建议使用浅色背景，文字使用深色，确保可读性

## 设计建议

### Header（横幅）
```
┌─────────────────────────────────────────────┐
│  [Logo] 应用名称 v1.0.0           [图标]    │
└─────────────────────────────────────────────┘
```

### Sidebar（侧边栏）
```
┌───────────┐
│           │
│   [Logo]  │
│           │
│  应用名称  │
│           │
│   图标    │
│           │
│           │
└───────────┘
```

## 快速开始

如果您现在不想创建自定义图片，可以：
1. 跳过图片配置，使用默认的 NSIS 样式
2. 在 `tauri.conf.json` 中注释掉图片相关配置

## 图片位置

将准备好的图片放置在本目录下：
```
src-tauri/installer/images/
├── installer-header.bmp
└── installer-sidebar.bmp
```
