# ToolHub 快速启动指南

欢迎使用 ToolHub！本指南将帮助你在5分钟内开始使用AI智能助手。

## 📋 前置要求

1. ✅ 已安装 ToolHub 应用
2. ⚠️ 需要 OpenAI 或兼容的 API 密钥（用于AI助手功能）

## 🚀 快速开始

### 第一步：配置AI API（必需）

1. 打开 ToolHub 应用
2. 点击左侧菜单栏的 **⚙️ 设置**
3. 选择 **AI 对话** 选项卡
4. 填写以下信息：

```
API Key: sk-xxxxxxxxxxxxxxxxxxxx
API Base URL: https://api.openai.com/v1
模型名称: gpt-3.5-turbo
```

5. 点击 **测试连接** 按钮验证配置
6. 看到"✓ 连接成功"后，点击 **保存**

> 💡 **提示**：如果你使用的是其他AI服务（如DeepSeek、通义千问等），请填写对应的API密钥和端点。

### 第二步：体验AI助手

#### 方式1：使用键盘快捷键 ⌨️

1. 按下 `Ctrl+K` (Mac用户按 `Cmd+K`)
2. 在弹出的对话框中输入：**"帮我写一篇关于Vue3的笔记"**
3. 按回车或点击发送
4. 等待AI处理（约2-3秒）
5. 确认AI提取的信息
6. 点击 **确认** 按钮
7. 完成！笔记已创建

#### 方式2：使用浮动按钮 🎈

1. 在应用右下角找到**蓝紫色的圆形按钮**
2. 点击它
3. 按照上面的步骤输入指令

#### 方式3：从工作台 🏠

1. 点击顶部的 **工作台**
2. 点击右上角的 **AI助手** 按钮
3. 按照上面的步骤输入指令

## 🔧 常见问题

### Q1: 提示"请先在设置中配置AI API"？

**A**: 需要先完成"第一步：配置AI API"。

### Q2: AI响应很慢？

**A**: 首次调用会较慢（冷启动），后续会变快。如果持续慢，检查网络连接。

### Q3: AI无法识别我的指令？

**A**: 尝试使用更清晰的表达，参考本文档的示例指令。

### Q4: 如何查看AI操作历史？

**A**: 在AI助手对话框底部点击"操作历史"按钮。

### Q5: 可以不使用AI助手吗？

**A**: 可以！所有功能都可以通过传统界面操作，AI助手只是快捷方式。

### Q6: 支持哪些AI模型？

**A**: 支持所有OpenAI兼容的API：
- ✅ OpenAI GPT系列 (gpt-3.5-turbo, gpt-4等)
- ✅ Anthropic Claude (通过兼容接口)
- ✅ 国内大模型 (DeepSeek, 通义千问, 文心一言等)

只要API格式兼容OpenAI即可。

## 💡 使用技巧

1. **善用快捷键**: `Ctrl+K` 是最快的唤起方式
2. **清晰表达**: "明天10点"比"明天上午"更准确
3. **批量操作**: 一次创建多个待办比分开创建更高效
4. **操作历史**: 查看历史可以学习如何更好地使用AI
5. **密码生成**: 不提供密码让AI自动生成更安全
