# Change: 重构首页仪表板并集成AI智能助手

## Why

当前首页的统计信息与实际模块数据脱节，展示的内容缺乏实用价值，无法为用户提供有效的工作概览。用户需要在不同模块间频繁切换才能完成日常任务，缺少统一的智能交互入口。

用户希望通过自然语言与系统交互，例如"帮我写一篇文档"、"明天上午10点提醒我开会"、"保存QQ账号密码"等，直接创建和管理各模块的数据，而不需要手动导航到对应模块。

## What Changes

### 首页仪表板重构
- 移除无意义的静态统计数据
- 添加实时数据展示：
  - 今日日期和天气（可选）
  - 今日待办事项（来自待办模块）
  - 今日日程安排（来自日历模块）
  - 最近编辑的笔记（来自笔记模块）
  - 常用网站快捷入口（来自书签模块）
  - 密码统计和安全提醒（来自密码管理模块）
- 优化UI布局，采用卡片式设计
- 所有数据卡片支持快速操作（如标记完成、快速编辑等）

### AI智能助手集成
- 添加全局AI对话框（浮动按钮或顶部快捷入口）
- 使用设置中配置的AI API（OpenAI/Claude/自定义API）
- 支持自然语言指令操作各模块：
  - **笔记模块**: "帮我写一个滕王阁序的md文档"、"创建一篇关于XX的笔记"
  - **日程模块**: "明天上午10点开会，提前5分钟提醒"、"下周三下午2点约会"
  - **密码模块**: "存储QQ密码，用户名XXX，密码XXX"、"保存网站账号"
  - **书签模块**: "收藏网站www.baidu.com"、"添加常用网站"
  - **待办模块**: "我有任务：第一步XX，第二步YY"、"创建购物清单"
- AI响应包含：
  - 意图识别（创建笔记/日程/密码等）
  - 参数提取（标题、时间、用户名、密码等）
  - 操作确认和结果反馈
  - 错误处理和重试机制
- 支持多轮对话和上下文保持
- 操作记录和可撤销功能

## Impact

### 受影响的模块
- **dashboard** (首页仪表板) - 完全重构
- **ai-assistant** (AI助手) - 新增功能
- **notes** (笔记) - 需要提供AI创建接口
- **calendar** (日程) - 需要提供AI创建接口
- **password-manager** (密码管理) - 需要提供AI创建接口
- **bookmarks** (书签) - 需要提供AI创建接口
- **todos** (待办) - 需要提供AI创建接口
- **settings** (设置) - AI API配置已存在，需确保可用

### 受影响的代码
- `src/views/Home.vue` - 首页组件完全重构
- `src/components/AIAssistant.vue` - 新建AI助手组件
- `src/views/NoteView.vue` - 添加 AI 创建笔记方法
- `src/views/Calendar.vue` - 添加 AI 创建日程方法
- `src/views/Passwords.vue` - 添加 AI 创建密码方法
- `src/views/Bookmarks.vue` - 添加 AI 创建书签方法
- `src/views/Todos.vue` - 添加 AI 创建待办方法
- `src/utils/ai/` - 新建AI服务工具类

### 数据库影响
- 无需新表，复用现有模块的数据表
- 可能需要添加 `ai_operations_log` 表记录AI操作历史（可选）

### 依赖
- 已有 OpenAI SDK (`openai` 包)
- 已有设置模块的 AI API 配置
- 需要增强 prompt 设计和意图识别逻辑

### 兼容性
- 向后兼容：原有模块功能不受影响
- 用户可选择使用传统UI或AI助手
- AI功能依赖于API配置，未配置时禁用或提示

### 用户体验改进
- 首页从"无用信息"变为"工作中心"
- AI助手提供极速操作入口
- 减少90%的点击和导航步骤
- 支持语音输入（未来扩展）
