## MODIFIED Requirements

### Requirement: 密码存储
系统应在 SQLite 数据库中安全地存储密码，并进行加密，提供密码强度分析和安全审计。

#### Scenario: 创建密码条目
- **WHEN** 用户点击"添加密码"
- **THEN** 系统打开表单，包含字段：标题、用户名、密码、网站、备注、过期时间
- **AND** 实时显示密码强度分析
- **AND** 保存时，将加密的密码存储到 SQLite 数据库
- **AND** 在密码列表中显示条目

#### Scenario: 查看密码列表
- **WHEN** 用户导航到密码管理器
- **THEN** 系统显示所有存储的密码
- **AND** 密码默认以掩码形式显示
- **AND** 显示标题、用户名和网站
- **AND** 显示密码强度指示器
- **AND** 显示过期状态

#### Scenario: 查看密码详情
- **WHEN** 用户点击密码条目
- **THEN** 系统显示完整详情
- **AND** 可以通过"显示密码"按钮显示密码
- **AND** 支持复制到剪贴板
- **AND** 显示密码强度分析
- **AND** 显示最后修改时间

#### Scenario: 编辑密码条目
- **WHEN** 用户在密码条目上点击"编辑"
- **THEN** 系统打开包含当前值的编辑表单
- **AND** 用户可以修改任何字段
- **AND** 实时显示密码强度
- **AND** 更改保存到数据库

#### Scenario: 删除密码条目
- **WHEN** 用户在密码条目上点击"删除"
- **THEN** 系统提示确认
- **AND** 确认后从数据库移除条目

#### Scenario: 搜索密码
- **WHEN** 用户在搜索框中输入
- **THEN** 系统按标题、用户名或网站过滤密码
- **AND** 实时更新结果
- **AND** 支持高级筛选（按强度、过期状态等）

## MODIFIED Requirements

### Requirement: 密码安全
系统应加密密码并提供安全的访问控制，包括密码强度分析和安全审计。

#### Scenario: 保存时加密密码
- **WHEN** 用户保存密码
- **THEN** 系统在存储前加密密码值
- **AND** 使用安全的加密算法
- **AND** 解密密钥安全管理

#### Scenario: 列表中掩码密码
- **WHEN** 密码在列表中显示
- **THEN** 密码值以 `••••••` 掩码显示
- **AND** 仅在用户明确请求时显示

#### Scenario: 密码强度分析
- **WHEN** 用户输入或编辑密码
- **THEN** 系统实时分析密码强度
- **AND** 显示强度等级（弱/中/强）
- **AND** 提供密码改进建议
- **AND** 检测常见弱密码模式

#### Scenario: 安全审计
- **WHEN** 用户查看安全审计
- **THEN** 系统检测弱密码
- **AND** 检测重复密码
- **AND** 检测过期密码
- **AND** 生成安全报告
- **AND** 提供修复建议

## ADDED Requirements

### Requirement: 密码导入导出
系统应支持密码数据的导入和导出功能，包括主流浏览器的密码导出格式。

#### Scenario: 导入通用 CSV 格式密码
- **WHEN** 用户选择导入密码并选择 CSV 文件
- **THEN** 系统解析 CSV 文件
- **AND** 验证导入数据格式
- **AND** 映射字段到系统格式（标题、用户名、密码、网站等）
- **AND** 显示导入结果和错误信息

#### Scenario: 导入 Google Chrome 密码
- **WHEN** 用户选择导入 Chrome 密码文件
- **THEN** 系统识别 Chrome 导出格式（CSV）
- **AND** 解析 Chrome 特定字段（name, url, username, password）
- **AND** 自动映射到系统格式
- **AND** 处理 Chrome 特有的数据格式
- **AND** 显示导入结果和统计信息

#### Scenario: 导入 Microsoft Edge 密码
- **WHEN** 用户选择导入 Edge 密码文件
- **THEN** 系统识别 Edge 导出格式（CSV）
- **AND** 解析 Edge 特定字段（name, url, username, password）
- **AND** 自动映射到系统格式
- **AND** 处理 Edge 特有的数据格式
- **AND** 显示导入结果和统计信息

#### Scenario: 导入 1Password/LastPass 格式
- **WHEN** 用户选择导入 1Password 或 LastPass 格式文件
- **THEN** 系统识别对应格式
- **AND** 解析加密或未加密的密码数据
- **AND** 自动映射字段
- **AND** 显示导入结果

#### Scenario: 导入数据验证
- **WHEN** 系统导入密码数据
- **THEN** 系统验证必填字段（至少需要密码）
- **AND** 检测重复密码条目
- **AND** 验证 URL 格式（如适用）
- **AND** 显示验证错误和警告
- **AND** 允许用户选择跳过错误或修复

#### Scenario: 导入结果处理
- **WHEN** 导入完成后
- **THEN** 系统显示导入统计（成功、失败、跳过数量）
- **AND** 显示详细错误列表（如有）
- **AND** 自动刷新密码列表
- **AND** 高亮显示新导入的密码

#### Scenario: 导出密码
- **WHEN** 用户选择导出密码
- **THEN** 系统支持导出为加密 JSON 格式
- **AND** 支持导出为 CSV 格式
- **AND** 需要主密码确认
- **AND** 支持选择性导出（按分类、标签等）

### Requirement: 密码过期管理
系统应支持密码过期提醒和管理。

#### Scenario: 设置密码过期时间
- **WHEN** 用户创建或编辑密码
- **THEN** 系统允许设置过期时间
- **AND** 过期前自动提醒

#### Scenario: 密码过期提醒
- **WHEN** 密码即将过期
- **THEN** 系统发送通知提醒
- **AND** 在列表中标记过期密码
