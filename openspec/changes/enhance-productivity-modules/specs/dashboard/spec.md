## MODIFIED Requirements

### Requirement: 首页仪表板
系统应提供一个功能丰富的首页仪表板，展示最近活动、数据统计、图表分析和所有模块的快速访问。

#### Scenario: 应用启动时查看仪表板
- **WHEN** 用户打开应用
- **THEN** 默认显示仪表板
- **AND** 展示最近笔记、密码、书签、今日待办和今日日程
- **AND** 显示数据统计卡片（笔记总数、密码数量、待办完成率、日程数量）

#### Scenario: 查看数据统计
- **WHEN** 用户在仪表板上
- **THEN** 系统显示各模块的数据统计卡片
- **AND** 包括笔记总数、密码数量、待办完成率、日程数量
- **AND** 统计卡片可点击跳转到对应模块

#### Scenario: 查看图表分析
- **WHEN** 用户在仪表板上
- **THEN** 系统显示数据可视化图表
- **AND** 包括待办完成趋势图（最近7天/30天）
- **AND** 包括笔记创建趋势图
- **AND** 包括日程分布饼图
- **AND** 图表支持交互（缩放、筛选）

#### Scenario: 查看最近笔记
- **WHEN** 用户在仪表板上
- **THEN** 系统显示最近修改的笔记列表
- **AND** 每个笔记显示标题、修改时间和预览
- **AND** 支持点击跳转到笔记详情

#### Scenario: 查看最近密码
- **WHEN** 用户在仪表板上
- **THEN** 系统显示最近访问的密码
- **AND** 密码以掩码形式显示以确保安全
- **AND** 显示密码强度指示器

#### Scenario: 查看常用书签
- **WHEN** 用户在仪表板上
- **THEN** 系统显示常用书签（按访问频率）
- **AND** 书签可点击在浏览器中打开
- **AND** 显示访问次数

#### Scenario: 查看今日待办
- **WHEN** 用户在仪表板上
- **THEN** 系统显示今日计划的待办事项
- **AND** 显示完成状态和进度
- **AND** 高亮显示过期待办

#### Scenario: 查看今日日程
- **WHEN** 用户在仪表板上
- **THEN** 系统显示今日的日程事件
- **AND** 显示事件时间和描述
- **AND** 按时间顺序排列

#### Scenario: 查看工作区概览
- **WHEN** 用户在仪表板上
- **THEN** 系统显示最近活动时间线
- **AND** 包括最近创建的笔记、待办、日程等
- **AND** 显示活动时间戳

#### Scenario: 自定义 Dashboard 布局
- **WHEN** 用户拖拽 Dashboard 卡片
- **THEN** 系统保存布局配置
- **AND** 下次打开时恢复自定义布局
- **AND** 支持显示/隐藏特定卡片

## ADDED Requirements

### Requirement: 快捷操作面板
系统应提供一个快捷操作面板，允许用户快速执行常用操作。

#### Scenario: 使用快捷操作
- **WHEN** 用户在仪表板上
- **THEN** 系统显示快捷操作按钮
- **AND** 包括"快速创建笔记"、"快速创建待办"等
- **AND** 点击后直接打开创建对话框
