## MODIFIED Requirements

### Requirement: 应用布局
系统应提供包含头部、侧边栏和内容区的三栏布局。

#### Scenario: 查看应用布局
- **WHEN** 应用启动
- **THEN** 系统在顶部显示头部栏
- **AND** 在左侧显示侧边栏
- **AND** 在右侧显示内容区
- **AND** 布局响应式且可调整大小

### Requirement: 头部栏
系统应提供包含应用名称和窗口控制的头部栏。

#### Scenario: 查看头部栏
- **WHEN** 应用显示时
- **THEN** 头部栏在左侧显示应用名称
- **AND** 在右侧显示窗口控制：置顶（始终置顶）、最小化、最大化、关闭
- **AND** 窗口控制使用原生样式

#### Scenario: 置顶窗口
- **WHEN** 用户点击置顶按钮
- **THEN** 系统切换"始终置顶"状态
- **AND** 启用时窗口保持在其他窗口之上

#### Scenario: 窗口控制
- **WHEN** 用户点击最小化
- **THEN** 窗口最小化到任务栏或托盘

- **WHEN** 用户点击最大化
- **THEN** 窗口在最大化和正常状态之间切换

- **WHEN** 用户点击关闭
- **THEN** 系统应用配置的关闭行为（询问/最小化/退出）

### Requirement: 侧边栏
系统应提供包含导航和设置的侧边栏。

#### Scenario: 查看侧边栏
- **WHEN** 应用显示时
- **THEN** 侧边栏在顶部显示首页
- **AND** 在中间显示功能模块（可折叠部分）
- **AND** 在底部显示设置

#### Scenario: 从侧边栏导航
- **WHEN** 用户点击首页
- **THEN** 系统导航到仪表板

- **WHEN** 用户点击模块（笔记、密码等）
- **THEN** 系统导航到该模块的页面

- **WHEN** 用户点击设置
- **THEN** 系统导航到设置页面

#### Scenario: 折叠侧边栏部分
- **WHEN** 用户点击部分标题
- **THEN** 部分展开或折叠
- **AND** 显示/隐藏模块项
- **AND** 状态持久化

#### Scenario: 折叠整个侧边栏
- **WHEN** 用户点击折叠按钮
- **THEN** 侧边栏折叠为仅图标模式
- **AND** 悬停或点击时展开

### Requirement: 内容区
系统应提供显示活动页面的内容区。

#### Scenario: 显示页面内容
- **WHEN** 用户导航到页面
- **THEN** 内容区显示该页面的组件
- **AND** 使用 Vue Router 进行导航
- **AND** 支持平滑过渡

#### Scenario: 响应式内容
- **WHEN** 窗口调整大小
- **THEN** 内容区调整宽度
- **AND** 保持最小可用宽度
- **AND** 组件适应可用空间

### Requirement: 启动动画和加载
系统应在应用启动时显示真实的加载动画，反映实际的配置和数据加载时间。

#### Scenario: 显示启动动画
- **WHEN** 应用启动
- **THEN** 系统显示启动画面（Splash Screen）
- **AND** 显示加载动画和进度指示
- **AND** 动画持续时间为实际加载配置和数据的时间

#### Scenario: 加载配置
- **WHEN** 应用启动
- **THEN** 系统从用户目录加载配置文件
- **AND** 显示"正在加载配置..."提示
- **AND** 等待配置加载完成后再继续

#### Scenario: 加载数据库
- **WHEN** 应用启动
- **THEN** 系统初始化 SQLite 数据库连接
- **AND** 显示"正在初始化数据库..."提示
- **AND** 等待数据库就绪后再继续

#### Scenario: 加载数据
- **WHEN** 应用启动
- **THEN** 系统加载待办、日程等数据
- **AND** 显示"正在加载数据..."提示
- **AND** 等待数据加载完成后再显示主界面

#### Scenario: 启动完成
- **WHEN** 所有加载任务完成
- **THEN** 系统关闭启动画面
- **AND** 显示主应用界面
- **AND** 启动动画总时长等于实际加载时间
