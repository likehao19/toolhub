# 变更：添加一体化生产力工具箱

## 原因
用户需要一个本地优先（Local First）的桌面生产力工具箱，整合笔记、密码、书签、待办、日程等功能于一体。当前模板项目仅提供基础功能演示，需要扩展为完整的生产级应用，支持插件化架构和 AI 辅助能力。

## 变更内容
- **ADDED**: 首页（Dashboard）模块，展示最近改动的笔记、密码、常用网站、今日待办、今日日程
- **ADDED**: 命令输入框，支持 `@` 符号快速创建内容（如 `@note` 创建笔记）
- **ADDED**: 笔记管理模块，支持 Markdown 编辑、文件夹/文件管理、图片/视频展示，本地存储
- **ADDED**: 密码管理模块，使用 SQLite 存储，支持增删改查
- **ADDED**: 网站管理模块，使用 SQLite 存储收藏网站，支持快速打开
- **ADDED**: 待办管理模块，支持子任务和进度跟踪，首页显示今日待办
- **ADDED**: 日程管理模块，展示中国日历和节假日，支持创建日程，首页显示今日日程
- **ADDED**: 设置模块，分为常规、外观、存储、AI 配置等子模块
- **ADDED**: 插件化架构，支持功能模块扩展
- **ADDED**: AI 辅助功能，集成大模型分析笔记，配置存储在设置中
- **ADDED**: 启动动画，显示真实的配置和数据加载时间，动画时长等于实际加载时间
- **ADDED**: 桌面通知系统，待办和日程时间到达时在桌面右下角弹出系统通知
- **MODIFIED**: 应用布局，添加头部（程序名称、窗口控制）、左侧菜单栏（首页、功能菜单、设置）、右侧内容区
