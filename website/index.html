为什么是 ToolHub

少一些工具切换，多一些专注时间。

我们没有重新发明笔记、日程或密码管理。 我们只是把它们好好地放在一起，并给它一种从容的节奏。

用一句话开始

不需要打开五个表单。打开命令面板，告诉它你想做什么，由它来理解、提取、确认、保存。

  • 明天 10 点产品评审，提前 5 分钟提醒
  • 保存 GitHub 密码，用户名 octocat
  • 记一条关于 Vue 3 的笔记

留在你的设备上

SQLite 本地数据库 + 本地加密。我们不上传你的笔记、密码或日程，也没有云端账号要登录。

  • 本地优先，无需注册
  • 密码字段独立加密
  • 支持完整数据备份与迁移

启动如一道闪光

由 Tauri 与 Rust 构建，安装包不到 10 MB，冷启动几乎察觉不到延迟。它属于你的桌面，而不是又一个浏览器标签。

  • 原生窗口与系统托盘
  • 单实例运行，唤起即聚焦
  • 亮 / 暗主题平滑切换
命令面板

说出来，比点开它更快。

按下 ⌘ K 唤起命令面板。 它知道"明天 10 点开会"是日程，"记一下今天读到的那段引文"是笔记，"保存 GitHub 密码"是密码条目。

每次写入之前，它会把识别结果摆在你面前——你点确认，它才落库。

还有这些

为日常的小事预留位置。

书签收藏

分组管理常用站点，自动获取图标，再不用记忆繁琐的网址。

局域网聊天

UDP 自动发现，二进制 WebSocket 协议，1 MB 分块并发，传文件如丝顺滑。

智能提醒

支持每日 / 每周 / 每月重复，提前 N 分钟通知，永远不让你错过下一件要事。

模板系统

预置周报、晨会、读书笔记等模板，一键创建，让重复劳动变成结构化输出。

密码强度审计

自动识别弱密码、重复密码、过期密码，附内置密码生成器。

系统托盘

关闭即最小化到托盘，托盘菜单可显示 / 隐藏 / 退出，单实例防止重复打开。

不止生活，也有工作

把开发工作流，也搬进同一处。

从调试接口到管 SDK，从 SQL 客户端到 SSH 终端 —— 把那些散落的小工具收拢成一组。

API 工作台

发请求、对结果、写文档；内置 Mock 服务，前后端联调不必再等。支持 REST / WebSocket / SSE。

SSH 终端

本机直接连远端服务器，多会话切换 + SFTP 文件传输，再也不用切窗口。

Git 工具集

仓库可视化、分支图谱、每日工作日报自动生成 —— 从 commit 历史里抽出真实的进度。

数据库客户端

SQLite 浏览 + Redis 客户端，建表查数据写 SQL，本地工具一站搞定，不用再装 Navicat。

SDK 多版本管理

Java / Node / Python 多版本切换，环境变量一键应用，告别 nvm / sdkman 的命令行折磨。

网络与编码工具

DNS / IP 查询、端口扫描、网速测试、Hex / Base64 / 正则 / 加解密 —— 日常那些零散小工具都在这一个抽屉里。
