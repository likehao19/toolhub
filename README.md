# ToolHub - 一体化生产力工具箱

📋 项目概述

ToolHub 是一个现代化的桌面应用，集成了笔记管理、日程规划、密码管理、书签收藏、待办事项等多种生产力工具，并配备了强大的AI智能助手，帮助你通过自然语言快速创建和管理内容。

  ---
🏗️ 技术栈

前端核心
- Vue 3 (3.5.13) - 渐进式框架
- Vite (6.0.3) - 极速构建工具
- Vue Router (4.6.4) - 路由管理
- Pinia (3.0.4) - 状态管理（支持持久化）
- Element Plus (2.13.0) - UI 组件库

后端核心
- Tauri 2 - Rust 桌面应用框架
- 9 个 Tauri 插件 - 提供文件系统、对话框、通知、剪贴板、Shell 等原生能力

✨ 核心功能

### 🏠 智能工作台

- **实时数据概览**：今日日期、待办事项、日程安排、最近笔记、常用网站、密码安全统计
- **卡片式设计**：优雅的界面布局，信息一目了然
- **快速操作**：直接在工作台完成标记待办、查看详情等操作

### 🤖 AI智能助手

- **自然语言交互**：用自然语言创建笔记、日程、密码、书签、待办
- **智能意图识别**：自动理解你的需求并提取关键信息
- **多种唤起方式**：
  - 键盘快捷键：`Ctrl+K` (Mac: `Cmd+K`)
  - 工作台AI按钮
  - 右下角浮动按钮
- **支持的操作**：
  - 📝 创建笔记："帮我写一篇关于Vue3的笔记"
  - 📅 创建日程："明天上午10点开会，提前5分钟提醒"
  - 🔐 保存密码："存储QQ密码，用户名123456"
  - 🔖 添加书签："收藏网站 www.baidu.com"
  - ✅ 创建待办："创建任务：学习Vue、学习Tauri"
- **操作确认机制**：每次操作前都会确认，避免误操作
- **操作历史记录**：查看所有AI助手执行的操作

### 📝 笔记管理

- Markdown编辑器，支持实时预览

🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run tauri dev    # 启动 Tauri + Vite 联调
```

### 打包应用

```bash
npm run tauri build  # 打包桌面应用（MSI/NSIS）
```

### 配置AI助手

1. 打开应用后，进入**设置 > AI 对话**
2. 填写以下信息：
   - **API Key**：你的OpenAI或兼容API的密钥
   - **API Base URL**：API端点（如 `https://api.openai.com/v1`）
   - **模型名称**：使用的模型（如 `gpt-3.5-turbo`）
3. 点击**测试连接**验证配置
4. 点击**保存**
5. 按 `Ctrl+K` 唤起AI助手开始使用！
