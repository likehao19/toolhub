# ToolHub 图标快速配置指南

## 🎯 目标

让 Windows 任务栏和系统托盘显示 ToolHub 工具箱图标。

## ⚡ 最快方法（3分钟）

### 步骤 1: 生成图标（选择一种方式）

**方式 A - 使用 Python 脚本（最快）**
```bash
pip install cairosvg pillow
python generate_icons.py
```

**方式 B - 使用在线工具**
1. 打开 `src-tauri/icons/tray-icon.svg`
2. 访问 https://www.aconvert.com/cn/image/svg-to-png/
3. 上传 SVG，设置尺寸：
   - 32x32 → 保存为 `32x32.png`
   - 128x128 → 保存为 `128x128.png`
   - 256x256 → 保存为 `128x128@2x.png`
4. 访问 https://www.aconvert.com/cn/icon/png-to-ico/
5. 上传其中一个 PNG，选择多尺寸（16,32,48,64,128,256）
6. 下载 → 保存为 `icon.ico`
7. 将所有文件复制到 `src-tauri/icons/` 覆盖旧文件

### 步骤 2: 测试效果

```bash
npm run tauri dev
```

查看：
- ✅ Windows 任务栏左下角
- ✅ 系统托盘右下角
- ✅ 应用内头部

## 🔍 常见问题

**Q: 为什么任务栏/托盘还是旧图标？**
A: Windows 会缓存图标。解决方法：
1. 完全退出应用（不是最小化）
2. 删除任务栏固定的图标
3. 重新启动应用
4. 如果还不行，重启 Windows Explorer：
   - 打开任务管理器
   - 找到 "Windows 资源管理器"
   - 右键 → 重新启动

**Q: 托盘图标不显示？**
A: 检查系统托盘设置：
- 设置 → 个性化 → 任务栏 → 系统托盘图标
- 确保 ToolHub 显示开关打开

**Q: 想用自己的图标？**
A: 
1. 准备 1024x1024 PNG（透明背景）
2. 替换 `src-tauri/icons/tray-icon.svg` 或直接生成 PNG
3. 按照上面步骤生成 ico 文件
4. 重启应用
