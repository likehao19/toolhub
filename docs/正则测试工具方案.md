# 正则测试工具方案

## 一、功能概述

提供实时正则表达式测试工具，支持输入正则 + 测试文本后即时高亮匹配、显示分组、执行替换预览，并内置常用正则模板，内嵌于工具箱开发工具分类中。

### 核心能力

| 能力 | 说明 |
|------|------|
| 实时匹配 | 输入正则和文本后实时执行匹配 |
| 匹配高亮 | 测试文本中用颜色背景高亮所有匹配项 |
| Flags 切换 | 支持 g、i、m、s、u 标志位复选切换 |
| 分组展示 | 显示每个匹配的完整文本、索引范围、捕获组 |
| 替换预览 | 输入替换字符串，实时预览替换结果 |
| 常用模板 | 预置邮箱、手机号、URL、IP 等常用正则，一键填入 |
| 错误提示 | 正则语法错误时实时显示错误信息 |

---

## 二、技术选型

| 方案 | 说明 | 选择 |
|------|------|------|
| 原生 `RegExp` | 浏览器内置，零依赖，支持所有标准标志位 | ✅ |
| `XRegExp` 库 | 扩展正则语法（命名组、Unicode 属性等） | ❌ 过重 |

**选择原生 RegExp**：ES2018+ 已支持命名捕获组、lookbehind、dotAll 等特性，无需第三方库。

## 五、核心实现

### 5.1 正则执行

```javascript
const executeRegex = () => {
  try {
    const re = new RegExp(pattern, flagsStr)
    const matches = []
    let m
    if (re.global) {
      while ((m = re.exec(testText)) !== null) {
        matches.push({ text: m[0], index: m.index, end: m.index + m[0].length, groups: m.slice(1) })
      }
    } else {
      m = re.exec(testText)
      if (m) matches.push({ text: m[0], index: m.index, end: m.index + m[0].length, groups: m.slice(1) })
    }
    return matches
  } catch (e) {
    return { error: e.message }
  }
}
```

### 5.2 匹配高亮

将文本按匹配位置拆分为交替的普通片段和高亮片段，使用 `v-html` 渲染：

```javascript
const highlightedHtml = computed(() => {
  if (!matches.length) return escapeHtml(testText)
  let html = '', lastEnd = 0
  for (const m of matches) {
    html += escapeHtml(testText.slice(lastEnd, m.index))
    html += `<mark class="hl hl-${colorIndex}">${escapeHtml(m.text)}</mark>`
    lastEnd = m.end
  }
  html += escapeHtml(testText.slice(lastEnd))
  return html
})
```
