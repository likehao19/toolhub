# ToolHub 工具箱开发指南

## 新增工具 Checklist

每次开发一个新的工具箱功能，需要修改以下 **7 个位置**。按顺序完成即可。

---

### 1. Rust 后端（如需要）

如果工具需要系统级能力（网络连接、进程管理、文件操作等），需要写 Rust 命令：

| 文件 | 操作 |
|------|------|
| `src-tauri/Cargo.toml` | 添加依赖 crate |
| `src-tauri/src/commands/{tool}.rs` | **新建** — 编写 `#[tauri::command]` 函数 |
| `src-tauri/src/commands/mod.rs` | 添加 `pub mod {tool};` |
| `src-tauri/src/lib.rs` | 在 `invoke_handler` 的 `generate_handler![]` 宏中注册每个命令 |

验证：`cd src-tauri && cargo check`

> 如果工具纯前端（如格式化、编码转换），跳过此步。

---

### 2. 数据库迁移（如需要）

如果工具需要持久化数据（连接配置、历史记录等）：

**文件：`src/utils/database.js`**

```javascript
// 1. 修改 targetVersion
const targetVersion = N  // 递增

// 2. 在 runMigrations() 末尾添加
if (currentVersion < N) {
  await migrateToVersionN(db)
}

// 3. 新增迁移函数
async function migrateToVersionN(db) {
  try {
    await db.execute(`CREATE TABLE IF NOT EXISTS ... `)
    await db.execute('CREATE INDEX IF NOT EXISTS ...')
  } catch (error) {
    if (!error.message?.includes('already exists')) {
      console.warn('Migration vN error:', error)
    }
  }
  try {
    await db.execute('INSERT OR IGNORE INTO schema_version (version) VALUES (N)')
  } catch (error) {
    try { await db.execute('UPDATE schema_version SET version = N WHERE version < N') } catch {}
  }
}
```

### 3. 前端工具函数

**文件：`src/utils/{toolName}Manager.js`** — 新建

封装 Tauri invoke 调用 + 业务逻辑：

```javascript
import { invoke } from '@tauri-apps/api/core'
import { getDatabase } from './database'

// 封装 invoke
export async function doSomething(args) {
  return await invoke('command_name', { ...args })
}

// 数据库 CRUD（如有）
export async function getItems() {
  const db = await getDatabase()
  return await db.select('SELECT * FROM ...')
}
```

## 项目技术栈

- **前端**：Vue 3 (Composition API `<script setup>`) + Element Plus + Vue Router
- **后端**：Rust + Tauri v2
- **数据库**：SQLite (tauri-plugin-sql)，递增版本迁移
- **国际化**：自定义 i18n，`t()` 函数
- **样式**：Scoped CSS + CSS 变量（`--bg-primary`、`--text-primary`、`--border-color`、`--accent-blue` 等）
- **通信**：前端 `invoke()` → Rust `#[tauri::command]`
- **存储**：SQLite 持久化 + localStorage 配置项
