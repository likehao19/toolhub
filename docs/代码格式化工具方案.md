# 代码格式化工具方案

## 一、功能概述

提供一站式代码格式化 & 美化工具，支持 **20+ 种语言**的语法高亮 + 一键格式化，内嵌于工具箱中。

### 核心能力

| 能力 | 说明 |
|------|------|
| 语法高亮 | 关键字、字符串、注释、数字等不同颜色渲染 |
| 一键格式化 | 缩进对齐、换行美化、括号规范 |
| 行号显示 | 左侧行号栏，支持行高亮 |
| 代码折叠 | 折叠/展开代码块 |
| 主题切换 | 跟随应用深色/浅色主题 |
| 语言检测 | 根据内容自动推断语言，支持手动选择 |
| 压缩/美化 | 双向切换：美化（Pretty）↔ 压缩（Minify） |

### 3.3 自动语言检测规则

```javascript
function detectLanguage(code) {
  const trimmed = code.trim()
  if (trimmed.startsWith('{') || trimmed.startsWith('[')) return 'json'
  if (trimmed.startsWith('<?xml') || trimmed.startsWith('<')) return 'xml' // or html
  if (/^(SELECT|INSERT|UPDATE|DELETE|CREATE|ALTER|DROP)\b/i.test(trimmed)) return 'sql'
  if (trimmed.startsWith('---') || /^\w+:\s/.test(trimmed)) return 'yaml'
  if (/^(package|import\s+java\.)/.test(trimmed)) return 'java'
  if (/^(func |package main)/.test(trimmed)) return 'go'
  if (/^(fn |use |mod |pub )/.test(trimmed)) return 'rust'
  if (/^(def |class |import |from )/.test(trimmed)) return 'python'
  if (/^#!\/(bin\/bash|usr\/bin\/env)/.test(trimmed)) return 'shell'
  if (/^(const |let |var |function |import |export )/.test(trimmed)) return 'javascript'
  return 'javascript' // fallback
}
```

## 五、核心实现

### 5.1 格式化调度器

```javascript
import * as prettier from 'prettier/standalone'
import prettierBabel from 'prettier/plugins/babel'
import prettierEstree from 'prettier/plugins/estree'
import prettierHtml from 'prettier/plugins/html'
import prettierCss from 'prettier/plugins/postcss'
import prettierMarkdown from 'prettier/plugins/markdown'
import prettierYaml from 'prettier/plugins/yaml'
import prettierGraphql from 'prettier/plugins/graphql'
import prettierJava from 'prettier-plugin-java'
import prettierPhp from '@prettier/plugin-php/standalone'
import { format as formatSQL } from 'sql-formatter'
import xmlFormatter from 'xml-formatter'

const PRETTIER_PLUGINS = [
  prettierBabel, prettierEstree, prettierHtml,
  prettierCss, prettierMarkdown, prettierYaml,
  prettierGraphql, prettierJava, prettierPhp,
]

/**
 * 格式化代码
 * @param {string} code - 源代码
 * @param {string} lang - 语言 key
 * @param {object} options - { tabWidth: 2|4 }
 */
export async function formatCode(code, lang, options = {}) {
  const config = LANGUAGES[lang]
  if (!config?.formatter) return code // 无格式化器，原样返回

  const tabWidth = options.tabWidth || 2

  switch (config.formatter) {
    case 'json':
      return JSON.stringify(JSON.parse(code), null, tabWidth)

    case 'prettier':
      return await prettier.format(code, {
        parser: config.parser,
        plugins: PRETTIER_PLUGINS,
        tabWidth,
        singleQuote: true,
        trailingComma: 'es5',
        printWidth: 100,
      })

    case 'sql':
      return formatSQL(code, {
        language: 'sql',
        tabWidth,
        keywordCase: 'upper',
      })

    case 'xml':
      return xmlFormatter(code, {
        indentation: ' '.repeat(tabWidth),
        collapseContent: true,
        lineSeparator: '\n',
      })

    default:
      return code
  }
}

/**
 * 压缩代码
 */
export function minifyCode(code, lang) {
  switch (lang) {
    case 'json':
      return JSON.stringify(JSON.parse(code))
    case 'xml':
      return xmlFormatter(code, { indentation: '', lineSeparator: '' })
    case 'css':
    case 'html':
    case 'javascript':
      // 简易压缩：移除多余空白和换行
      return code.replace(/\s+/g, ' ').replace(/\s*([{}();,:])\s*/g, '$1').trim()
    default:
      return code
  }
}
```

## 九、与其他工具箱工具对比

本工具与 JSON 格式化等单一工具不同：

- **统一入口**：一个页面处理所有语言，无需每种语言单独做一个工具
- **专业编辑器**：CodeMirror 6 提供接近 IDE 的体验
- **可扩展**：新增语言只需添加配置，无需改 UI
