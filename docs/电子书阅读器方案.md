# 电子书管理与阅读器方案

## 一、功能概述

| 能力 | 说明 |
|------|------|
| 书架管理 | 书籍卡片网格展示，封面/书名/作者/阅读进度，支持分类、搜索、排序 |
| 书籍导入 | 支持导入 EPUB / PDF / TXT 格式，拖拽导入或文件选择对话框 |
| 存储管理 | 用户配置存储目录，导入时自动复制到指定位置，统一管理 |
| 仿真翻页 | CSS 3D Transform 模拟真实书本翻页效果（翻转、卷曲、阴影） |
| 滑动翻页 | 备选翻页模式：左右滑动切换页面（触屏友好） |
| 滚动模式 | 第三种阅读模式：连续滚动长页面，适合快速浏览 |
| 阅读进度 | 自动记录当前阅读位置，下次打开自动恢复 |
| 书签标注 | 添加书签、文字高亮标注、笔记批注 |
| 目录导航 | 解析 EPUB/PDF 内置目录，快速跳转章节 |
| 阅读设置 | 字体、字号、行高、主题色（护眼/暗色/自定义）、页边距 |
| 全文搜索 | 搜索书籍内文字内容，结果高亮跳转 |
| 阅读统计 | 记录阅读时长、页数、完成率 |

## 二、技术选型

### EPUB 解析

| 方案 | 包大小 | 渲染质量 | 流式/分页 | 交互能力 | 选择 |
|------|--------|---------|----------|---------|------|
| **epubjs** | ~180 KB | 优 | 均支持 | 选择/高亮/批注 | ✅ |
| readium | ~500 KB | 优 | 均支持 | 丰富 | ❌ 体积大 |
| 手动解析 zip+xhtml | 0 | 一般 | 需自建 | 有限 | ❌ 工作量大 |

选择 **`epubjs`**（epub.js v0.3+），理由：
- 成熟稳定，GitHub 6k+ star，社区活跃
- 内置分页引擎、目录解析、搜索、高亮/标注 API
- 支持自定义主题和字体注入
- 渲染到 iframe，天然隔离样式

### PDF 渲染

| 方案 | 包大小 | 渲染方式 | 文字选择 | 选择 |
|------|--------|---------|---------|------|
| **pdfjs-dist** | ~400 KB | Canvas/SVG | ✅ 文字层 | ✅ |
| vue-pdf-embed | ~50 KB | 封装 pdfjs | ✅ | ❌ 灵活性不足 |
| 原生 webview | 0 | 系统 PDF | ✅ | ❌ 无法定制 |

选择 **`pdfjs-dist`**（Mozilla PDF.js），理由：
- 业界标准 PDF 渲染引擎，兼容性最佳
- 支持文字层（可选择/搜索）、缩放、旋转
- 支持逐页渲染，可结合翻页动画

### TXT 解析

原生实现，无需第三方库：
- 按字符数/段落自动分页
- 支持 UTF-8 / GBK 编码检测（复用项目已有逻辑）
- 排版引擎：首行缩进、段落间距、两端对齐

### 翻页动画

| 方案 | 依赖 | 效果 | 性能 | 选择 |
|------|------|------|------|------|
| **CSS 3D Transform** | 无 | 仿真翻页 | GPU 加速，流畅 | ✅ |
| turn.js | 50 KB + jQuery | 逼真 | 依赖 jQuery | ❌ |
| Canvas 绘制 | 无 | 高度自定义 | CPU 密集 | ❌ |

选择 **CSS 3D Transform** 纯 CSS 方案：
- 零依赖，利用 `perspective`、`rotateY`、`backface-visibility`
- GPU 硬件加速，60fps 流畅翻页
- 支持触摸拖拽跟手翻页

### 数据存储

复用项目已有的 SQLite（`src/utils/database.js`），新增 `books`、`bookmarks`、`highlights` 表。

```sql
-- 书籍信息表
CREATE TABLE IF NOT EXISTS books (
  id          INTEGER PRIMARY KEY AUTOINCREMENT,
  title       TEXT NOT NULL,               -- 书名
  author      TEXT DEFAULT '',             -- 作者
  cover       TEXT DEFAULT '',             -- 封面图片路径或 base64
  format      TEXT NOT NULL,               -- epub / pdf / txt
  file_path   TEXT NOT NULL,               -- 文件存储路径
  file_size   INTEGER DEFAULT 0,           -- 文件大小 (bytes)

  description TEXT DEFAULT '',             -- 简介
  language    TEXT DEFAULT '',             -- 语言
  publisher   TEXT DEFAULT '',             -- 出版社
  isbn        TEXT DEFAULT '',             -- ISBN
  created_at  TEXT DEFAULT (datetime('now')),
  updated_at  TEXT DEFAULT (datetime('now'))
);

-- 阅读进度表
CREATE TABLE IF NOT EXISTS reading_progress (
  id          INTEGER PRIMARY KEY AUTOINCREMENT,
  book_id     INTEGER NOT NULL,
  current_page INTEGER DEFAULT 0,          -- 当前页码
  current_cfi  TEXT DEFAULT '',            -- EPUB CFI 定位
  progress    REAL DEFAULT 0,              -- 进度百分比 0-100
  total_time  INTEGER DEFAULT 0,           -- 累计阅读时长(秒)
  last_read   TEXT DEFAULT (datetime('now')),
  FOREIGN KEY (book_id) REFERENCES books(id) ON DELETE CASCADE
);

-- 书签表
CREATE TABLE IF NOT EXISTS book_bookmarks (
  id          INTEGER PRIMARY KEY AUTOINCREMENT,
  book_id     INTEGER NOT NULL,
  page        INTEGER DEFAULT 0,           -- 页码
  cfi         TEXT DEFAULT '',             -- EPUB CFI
  title       TEXT DEFAULT '',             -- 书签标题
  color       TEXT DEFAULT '#409eff',      -- 书签颜色
  created_at  TEXT DEFAULT (datetime('now')),
  FOREIGN KEY (book_id) REFERENCES books(id) ON DELETE CASCADE
);

-- 高亮标注表
CREATE TABLE IF NOT EXISTS book_highlights (
  id          INTEGER PRIMARY KEY AUTOINCREMENT,
  book_id     INTEGER NOT NULL,
  page        INTEGER DEFAULT 0,
  cfi_range   TEXT DEFAULT '',             -- EPUB CFI 范围
  text        TEXT NOT NULL,               -- 高亮文字
  note        TEXT DEFAULT '',             -- 批注内容
  color       TEXT DEFAULT '#ffeb3b',      -- 高亮颜色
  created_at  TEXT DEFAULT (datetime('now')),
  FOREIGN KEY (book_id) REFERENCES books(id) ON DELETE CASCADE
);
```

### 5.2 ebookManager.js — 核心逻辑

```javascript
import { getDatabase } from './database'
import { readFile, writeFile, copyFile, checkExists, createDir } from './tauri/fs'
import { openFile } from './tauri/dialog'

const SUPPORTED_FORMATS = ['epub', 'pdf', 'txt']
const BOOKS_DIR_KEY = 'ebook_storage_path'

/**
 * 获取书籍存储目录
 */
export function getBooksDir() {
  return localStorage.getItem(BOOKS_DIR_KEY) || ''
}

/**
 * 导入书籍
 * 1. 打开文件选择对话框
 * 2. 解析元数据（书名、作者、封面）
 * 3. 复制到存储目录
 * 4. 写入数据库
 */
export async function importBook(filePath) {
  const ext = filePath.split('.').pop().toLowerCase()
  if (!SUPPORTED_FORMATS.includes(ext)) {
    throw new Error(`不支持的格式: .${ext}`)
  }

  // 解析元数据
  const metadata = await parseMetadata(filePath, ext)

  // 复制到存储目录
  const booksDir = getBooksDir()
  if (booksDir) {
    await createDir(booksDir, { recursive: true })
    const destPath = `${booksDir}/${metadata.title || 'untitled'}.${ext}`
    await copyFile(filePath, destPath)
    metadata.file_path = destPath
  } else {
    metadata.file_path = filePath
  }

  // 写入数据库
  const db = await getDatabase()
  const result = await db.execute(
    `INSERT INTO books (title, author, cover, format, file_path, file_size, total_pages, description)
     VALUES (?, ?, ?, ?, ?, ?, ?, ?)`,
    [metadata.title, metadata.author, metadata.cover, ext,
     metadata.file_path, metadata.file_size, metadata.total_pages, metadata.description]
  )

  return { id: result.lastInsertId, ...metadata }
}

/**
 * 解析书籍元数据
 */
async function parseMetadata(filePath, format) {
  if (format === 'epub') return parseEpubMetadata(filePath)
  if (format === 'pdf') return parsePdfMetadata(filePath)
  return parseTxtMetadata(filePath)
}

/**
 * 解析 EPUB 元数据
 */
async function parseEpubMetadata(filePath) {
  const bytes = await readFile(filePath)
  const book = ePub(bytes.buffer)
  await book.ready

  const metadata = book.packaging.metadata
  let cover = ''
  try {
    cover = await book.coverUrl()  // base64 封面
  } catch { /* 无封面 */ }

  return {
    title: metadata.title || getFileName(filePath),
    author: metadata.creator || '',
    cover,
    description: metadata.description || '',
    total_pages: book.locations?.length() || 0,
    file_size: bytes.byteLength
  }
}

/**
 * 获取所有书籍
 */
export async function getAllBooks() {
  const db = await getDatabase()
  return await db.select(`
    SELECT b.*, rp.progress, rp.last_read, rp.current_page
    FROM books b
    LEFT JOIN reading_progress rp ON b.id = rp.book_id
    ORDER BY COALESCE(rp.last_read, b.created_at) DESC
  `)
}

/**
 * 更新阅读进度
 */
export async function updateProgress(bookId, { currentPage, currentCfi, progress }) {
  const db = await getDatabase()
  await db.execute(`
    INSERT INTO reading_progress (book_id, current_page, current_cfi, progress, last_read)
    VALUES (?, ?, ?, ?, datetime('now'))
    ON CONFLICT(book_id) DO UPDATE SET
      current_page = excluded.current_page,
      current_cfi = excluded.current_cfi,
      progress = excluded.progress,
      last_read = datetime('now')
  `, [bookId, currentPage, currentCfi || '', progress])
}

/**
 * 删除书籍
 */
export async function deleteBook(bookId, deleteFile = false) {
  const db = await getDatabase()
  if (deleteFile) {
    const [book] = await db.select('SELECT file_path FROM books WHERE id = ?', [bookId])
    if (book?.file_path) {
      try { await removeFile(book.file_path) } catch { /* ignore */ }
    }
  }
  await db.execute('DELETE FROM books WHERE id = ?', [bookId])
}
```

### 5.3 PageFlip.vue — 3D 翻页核心

```javascript
/**
 * 翻页动画核心逻辑
 *
 * 结构：
 *   .book-container (perspective: 1500px)
 *     .page.left   — 当前左页（静止）
 *     .page.right  — 当前右页（翻页时从右→左旋转）
 *     .page.next   — 下一页（翻页完成后显示）
 *
 * 翻页过程：
 *   1. 用户点击右侧/滑动 → 触发翻页
 *   2. .page.right 应用 transform: rotateY(-180deg)
 *   3. 同时显示 .page.right 背面（下一页左页内容）
 *   4. 动画完成 → 更新页码，重置 DOM 位置
 *
 * 拖拽跟手：
 *   - mousedown/touchstart 记录起始位置
 *   - mousemove/touchmove 计算角度 = (dx / pageWidth) * 180
 *   - mouseup/touchend 判断是否超过阈值(50%)，决定翻页或回弹
 */
```

关键 CSS：


### 5.4 EPUB 渲染集成

```javascript
import ePub from 'epubjs'

function renderEpub(container, bookData) {
  const book = ePub(bookData)
  const rendition = book.renderTo(container, {
    width: '100%',
    height: '100%',
    spread: 'always',         // 双页显示
    flow: 'paginated',        // 分页模式
    manager: 'continuous'
  })

  // 应用自定义主题
  rendition.themes.register('custom', {
    body: {
      'font-family': config.fontFamily,
      'font-size': config.fontSize + 'px',
      'line-height': config.lineHeight,
      'color': theme.textColor,
      'background': theme.bgColor,
    }
  })
  rendition.themes.select('custom')

  // 监听翻页
  rendition.on('relocated', (location) => {
    updateProgress(bookId, {
      currentCfi: location.start.cfi,
      progress: book.locations.percentageFromCfi(location.start.cfi) * 100
    })
  })

  // 高亮文字
  rendition.on('selected', (cfiRange, contents) => {
    // 弹出标注菜单
    showAnnotationMenu(cfiRange, contents)
  })

  return { book, rendition }
}
```

### 5.5 PDF 渲染集成

```javascript
import * as pdfjsLib from 'pdfjs-dist'

async function renderPdfPage(pdfDoc, pageNum, canvas) {
  const page = await pdfDoc.getPage(pageNum)
  const viewport = page.getViewport({ scale: 1.5 })

  canvas.width = viewport.width
  canvas.height = viewport.height

  await page.render({
    canvasContext: canvas.getContext('2d'),
    viewport
  }).promise

  // 渲染文字层（可选择/搜索）
  const textContent = await page.getTextContent()
  const textLayer = document.createElement('div')
  textLayer.className = 'pdf-text-layer'
  pdfjsLib.renderTextLayer({
    textContent,
    container: textLayer,
    viewport,
    textDivs: []
  })

  return textLayer
}
```

```javascript
/**
 * 将纯文本按页面大小分页
 * @param {string} text - 全文内容
 * @param {object} config - { width, height, fontSize, lineHeight, margin }
 * @returns {string[]} 每页的文本内容
 */
export function paginateText(text, config) {
  const { width, height, fontSize, lineHeight, margin } = config
  const contentWidth = width - margin * 2
  const contentHeight = height - margin * 2
  const charsPerLine = Math.floor(contentWidth / fontSize)
  const linesPerPage = Math.floor(contentHeight / (fontSize * lineHeight))

  const paragraphs = text.split(/\n+/)
  const pages = []
  let currentPage = []
  let currentLines = 0

  for (const para of paragraphs) {
    // 段落首行缩进 2 字符
    const indented = '　　' + para.trim()
    const lines = wrapText(indented, charsPerLine)

    for (const line of lines) {
      if (currentLines >= linesPerPage) {
        pages.push(currentPage.join('\n'))
        currentPage = []
        currentLines = 0
      }
      currentPage.push(line)
      currentLines++
    }
    // 段落间空行
    currentLines++
  }

  if (currentPage.length) {
    pages.push(currentPage.join('\n'))
  }

  return pages
}
```
