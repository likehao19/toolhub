**步骤 1: 准备 ToolHub 图标**

我已经在 `src-tauri/icons/tray-icon.svg` 创建了 ToolHub 工具箱图标。你需要将它转换为适合 Windows 的格式：

1. **转换 SVG 为 PNG (32x32)**
   - 访问：https://www.aconvert.com/cn/image/svg-to-png/
   - 上传 `src-tauri/icons/tray-icon.svg`
   - 设置尺寸：32x32 像素
   - 下载并保存为 `32x32.png`

2. **转换 SVG 为 PNG (128x128)**
   - 同样的网站，尺寸设置为 128x128
   - 下载并保存为 `128x128.png`

3. **转换 SVG 为 PNG (256x256)**
   - 同样的网站，尺寸设置为 256x256
   - 下载并保存为 `128x128@2x.png`

4. **创建 .ico 文件（Windows 任务栏图标）**
   - 访问：https://www.aconvert.com/cn/icon/png-to-ico/
   - 上传刚才生成的 PNG 图片
   - 选择包含多个尺寸：16, 32, 48, 64, 128, 256
   - 下载并保存为 `icon.ico`

5. **替换图标文件**
   - 将生成的所有文件复制到 `src-tauri/icons/` 目录
   - 覆盖原有文件：
     - `32x32.png`
     - `128x128.png`
     - `128x128@2x.png`
     - `icon.ico`

**步骤 2: 更新托盘图标代码**

托盘图标目前使用的是 `app.default_window_icon()`，这会使用 `icon.ico`。
如果你想使用专门的托盘图标，可以修改 `src-tauri/src/setup.rs`：

```rust
// 在文件顶部添加
use tauri::image::Image;
use std::path::PathBuf;

// 在 setup_tray 函数中，替换：
TrayIconBuilder::new()
    .icon(app.default_window_icon().unwrap().clone())  // 删除这行
    .icon(load_tray_icon()?)  // 添加这行
    .menu(&menu)
    // ... 其他代码不变

// 添加新函数：
fn load_tray_icon() -> Result<Image, Box<dyn std::error::Error>> {
    // Windows 使用 ico 格式
    #[cfg(target_os = "windows")]
    {
        let icon_path = PathBuf::from("icons/icon.ico");
        let icon_bytes = include_bytes!("../icons/icon.ico");
        Ok(Image::from_bytes(icon_bytes)?)
    }
    
    // macOS 使用 png 格式
    #[cfg(target_os = "macos")]
    {
        let icon_bytes = include_bytes!("../icons/32x32.png");
        Ok(Image::from_bytes(icon_bytes)?)
    }
    
    // Linux 使用 png 格式
    #[cfg(target_os = "linux")]
    {
        let icon_bytes = include_bytes!("../icons/32x32.png");
        Ok(Image::from_bytes(icon_bytes)?)
    }
}
```

## 📝 如何更换图标

### 方法一：使用 Python 脚本自动生成（最简单，推荐）

如果你安装了 Python：

```bash
# 1. 安装依赖
pip install cairosvg pillow

# 2. 运行脚本
python generate_icons.py
```

脚本会自动：
- 从 `src-tauri/icons/tray-icon.svg` 生成所有需要的 PNG 和 ICO 文件
- 包括 Windows 任务栏图标 (icon.ico)
- 包括系统托盘图标 (32x32.png)
- 自动覆盖旧图标

完成后重新运行 `npm run tauri dev` 即可看到新图标！

### 方法二：使用在线工具生成（推荐）

1. **准备原始图标**
   - 准备一张高质量的 PNG 图片（建议 1024x1024 像素）
   - 背景最好是透明的
   - 格式：PNG
   - **或者使用我已经创建的 `src-tauri/icons/tray-icon.svg`**

2. **生成 Windows .ico 文件（⭐ 最重要）**
   - 访问：https://www.aconvert.com/cn/icon/png-to-ico/
   - 上传你的 PNG 图片
   - 选择图标尺寸：16x16, 32x32, 48x48, 64x64, 128x128, 256x256
   - 下载生成的 .ico 文件，重命名为 `icon.ico`

3. **生成 macOS .icns 文件**
   - 访问：https://cloudconvert.com/png-to-icns
   - 上传你的 PNG 图片
   - 转换并下载，重命名为 `icon.icns`

4. **生成各种尺寸的 PNG**
   - 访问：https://www.iloveimg.com/zh-cn/resize-image
   - 分别生成：
     - 32x32.png
     - 128x128.png
     - 128x128@2x.png (256x256)
     - icon.png (512x512 或 1024x1024)

5. **替换文件并测试**
   - 将生成的所有文件复制到 `src-tauri/icons/` 目录
   - 覆盖原有文件
   - **重启应用**：`npm run tauri dev`
   - **检查效果**：
     - ✅ Windows 任务栏显示新图标
     - ✅ 系统托盘显示新图标
     - ✅ 应用内头部显示新图标

### 方法三：使用命令行工具

#### Windows 用户
```bash
# 安装 ImageMagick
choco install imagemagick

# 转换 PNG 到 ICO
magick convert icon.png -define icon:auto-resize=256,128,64,48,32,16 icon.ico
```

#### macOS 用户
```bash
# 创建 iconset 文件夹
mkdir icon.iconset

# 生成不同尺寸（假设你有一个 icon.png）
sips -z 16 16     icon.png --out icon.iconset/icon_16x16.png
sips -z 32 32     icon.png --out icon.iconset/icon_16x16@2x.png
sips -z 32 32     icon.png --out icon.iconset/icon_32x32.png
sips -z 64 64     icon.png --out icon.iconset/icon_32x32@2x.png
sips -z 128 128   icon.png --out icon.iconset/icon_128x128.png
sips -z 256 256   icon.png --out icon.iconset/icon_128x128@2x.png
sips -z 256 256   icon.png --out icon.iconset/icon_256x256.png
sips -z 512 512   icon.png --out icon.iconset/icon_256x256@2x.png
sips -z 512 512   icon.png --out icon.iconset/icon_512x512.png
sips -z 1024 1024 icon.png --out icon.iconset/icon_512x512@2x.png

# 生成 .icns 文件
iconutil -c icns icon.iconset
```

## 🎨 图标设计建议

1. **尺寸**：原图至少 1024x1024 像素
2. **格式**：PNG 格式，背景透明
3. **风格**：简洁明了，易于识别
4. **颜色**：使用企业品牌色
5. **细节**：在小尺寸下也要清晰可辨
